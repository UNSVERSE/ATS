The Theory Behind the ATS

1. Electrical Interlocking (The Safety Rule)

The most critical rule of an ATS is that both contactors must never be closed at the same time. If city power and generator power collide, it causes a catastrophic short circuit.

To prevent this, we use Electrical Interlocking. The control wire for the Grid Contactor is routed through a Normally Closed (NC) auxiliary switch on the Generator Contactor. Likewise, the Generator's control wire is routed through the Grid Contactor's NC switch. If one contactor is physically pulled shut, it forces its NC switch open, instantly cutting any possibility of power reaching the opposite contactor.

2. Grid Priority

What happens if the Grid and the Generator both have power at the exact same moment? The system must make a choice. By wiring the main sensing Relay directly to the Grid Phase, we establish Grid Priority.

When the Grid is alive, the Relay activates and physically breaks the pathway to the Generator's timer. The ATS will completely ignore the Generator until the Grid voltage drops back to zero.

3. Holding Circuits (Manual Mode)

When using Push Buttons in Manual Mode, pressing the "Start" button connects the circuit, but letting go breaks it. To keep the contactor running after you release the button, we use a Holding Circuit (or Latching Circuit).

We wire a Normally Open (13/14) auxiliary block on the contactor in parallel with the push button. When the button is pressed, the contactor closes, which also closes the 13/14 block. Power now flows through the 13/14 block, feeding the contactor its own power so it stays latched on indefinitely until the selector switch cuts the main control power.
